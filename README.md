# rotate-image.yazi

## Requirements
[imagemagick](https://github.com/ImageMagick/ImageMagick)

## Installation

```sh
ya pack -a tasnimAlam/rotate-image
```

## Usage

Add this to your `~/.config/yazi/keymap.toml`:

```toml
[[manager.prepend_keymap]]
on   = ["R"]
run  = "plugin rotate-image"
desc = "Rotate images"
```
